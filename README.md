```
第一章 营养学基础  1
一、概述  1
    1.1 食物成分  1
        1.1.1 营养素种类及分类  1
        1.1.2 水及其他膳食成分  1
    1.2 人体营养需要  2
        1.2.1 营养素的代谢及生理功能  2
        1.2.2 营养对人体构成的影响  2
二、营养素的代谢及生理功能  1
    2.1 营养素的代谢  1
    2.2 营养素的生理功能  2
```
